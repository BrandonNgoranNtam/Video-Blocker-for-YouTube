API Key

Chave de API é necessária para conseguir informações sobre vídeos, como tags e informações sobre canais.
Para conseguir uma chave e ativá-la, siga os passos da seção abaixo:

Como conseguir uma Chave de API

Você vai precisar de uma conta Google, para que possa solicitar as credenciais da API, tenha uma preparada.

Acesse o Console de APIs e clique em Criar Projeto, dê um nome para o projeto (Exemplo: Youtube Blocker) e clique em Criar.

Com o projeto que você criou selecionado, clique em + ATIVAR APIS E SERVIÇOS,
pesquise na barra de pesquisa por Youtube Data API e clique em Youtube Data API v3.

Na próxima pagina, clique no botão ATIVAR.
Isso vai redirecionar ao painel do YouTube Data API v3, em que vai solicitar que crie credenciais para que use a API,
clique em Criar Credenciais.

Na lista dropdown, selecione Youtube Data API v3.
Depois, abaixo, marque Dados públicos e clique em Próxima

Você conseguiu a chave! Copie ela e clique em Concluir

Pronto! Agora que suas credenciais foram copiadas, cole na caixa das opções e Salve suas configurações.
