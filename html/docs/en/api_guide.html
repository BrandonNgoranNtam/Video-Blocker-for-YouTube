API Key

API key is necessary to get information about youtube videos, such as tags, and channel data.
A guide with detailed instructions and screenshots of how to get an API key below:

How to Get One

You will need a google account to request an API key, make sure you have one.

Go to Google Developers Console and click Create Project, give the project a name (Example: Youtube Blocker) and click Create.

With the project you just created selected, click + ENABLE APIS AND SERVICES , and search for Youtube Data API, click on Youtube Data API v3.

On the YouTube Data API v3 page, click Enable.
This will redirect you to the overview page, click Create Credentials

In the dropdown, select Youtube Data API v3.
Then check Public data and click Next

You got your key! Copy it and click Done

Now you can paste the key you just copied to the input in the options, then Save it.
